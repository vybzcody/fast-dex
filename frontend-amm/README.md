# FastDEX Frontend

React + TypeScript frontend for the FastDEX micro-pools DEX on Linera.

## Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **GraphQL** - API communication with Linera service

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- **Swap Interface**: Trade tokens using CPMM pools
- **Pool Management**: Create and manage liquidity pools
- **Balance Tracking**: View token balances
- **Price Estimation**: Real-time swap quotes

## GraphQL Integration

The frontend connects to the Linera service GraphQL endpoint to:
- Query pool data and user balances
- Execute swaps and liquidity operations
- Estimate trade outputs

See the main project README for full API documentation.
